# dockerにてGitHub接続方法

## 前提条件
```sh
dnf install -y git openssh-clients

git --version

git config --global init.defaultBranch main
```

## Gitの設定

```sh
git config --global user.name {gitユーザー名}

git config --global user.email {git_email}

git config --global core.autocrlf input

git config --global --list
```

## GitHub連携
```sh
ssh-keygen -t ed25519 -C "your_email@example.com"

cat id_ed25519.pub
```

- https://github.com/settings/keys

- 「New SSH key」から鍵を登録
Title: Ubuntu等のわかりやすい名前を登録する。
Key Type: Authentication Key
Key : 公開鍵をクリップボードにコピーして、Githubに鍵を登録する ↓

## SSH確認
```sh
ssh -T git@github.com
```

## gitリモート接続

```sh
## リモートのリポジトリを登録
git remote add origin https://github.com/{リポジトリ}.git

## コミット
git add .
git commit -m "first commit"

git branch -M main

## URLをSSH形式に変更 ※「https://github.com/」は除く
git remote set-url origin git@github.com:{リポジトリ}.git


## 確認
git remote -v

git push -u origin main

```
