# 仮想環境の設定方法

```sh
## プロジェクトディレクトリへ移動
cd my-project

## 仮想環境を作成
python -m venv venv

## 仮想環境を有効化（activate）
source venv/Scripts/activate

## 仮想環境の無効化
deactivate

## requirements.txt を使った管理
pip freeze > requirements.txt

## 別環境で再現
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

## 仮想環境を作り直す
deactivate
rm -rf venv
python -m venv venv
```
